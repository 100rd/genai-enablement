## Human Impact Metrics

| Metric | Measurement Method |
|--------|-------------------|
| On-call burden | Survey + page count per rotation |
| Root cause analysis time | Time from alert to identified cause |
| Knowledge sharing | Playbooks created, reuse count |
| Team satisfaction | Quarterly survey (1-10 scale) |
| AI tool adoption rate | Active users / total team size |

## Per-Engagement Success Criteria

An engagement is successful when:
- [ ] Team can independently use GenAI tools after enablement
- [ ] Measurable improvement in at least 3 key metrics
- [ ] Documented playbook created for reuse
- [ ] No increase in incidents or security risks
- [ ] Positive team satisfaction scores (7+/10)

## Program-Level Success Criteria

The overall program succeeds when:
- [ ] 5+ successful team engagements completed
- [ ] 10+ reusable playbooks created
- [ ] 50%+ average MTTR reduction across teams
- [ ] 30%+ average deployment frequency increase
- [ ] Self-service adoption by teams not directly engaged

## Reporting Cadence

- **Weekly**: Key metric snapshots per active engagement
- **Monthly**: Program-level dashboard update
- **Quarterly**: Full ROI report with case studies
